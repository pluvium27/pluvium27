### 兴趣项目

**潜进/HeurAMS** - 启发式辅助记忆调度器

> 为问答, 记忆, 理解型知识设计的开放源代码辅助记忆软件, 提供动态规划的优化记忆方案  
包含以下子项目:  
 - **HeurAMS**: 核心与基于 Textual 的 TUI 界面  
 - **HeurAMS-F**: 基于 Flutter 的 现代用户界面  
 - **HeurSync**: 多用户数据同步服务器实现  
 - **HeurRepo**: 文档源实现
